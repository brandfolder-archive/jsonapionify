  • ¶

    Attributes may contain any valid JSON value. Complex data structures involving JSON objects and arrays are allowed as attribute values. However, any object that constitutes or is contained in an attribute MUST NOT contain a relationships or links member, as those members are reserved by this specification for future use.

        must_not_contain! :relationships, :links, deep: true
